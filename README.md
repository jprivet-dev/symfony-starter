## Installation

### 1. Clone the project

```shell
git clone git@github.com:MY_USERNAME/my-project.git
cd my-project
```

### 2. Start and install

```shell
make install
```

### Then with Makefile...

```shell
make start  # Start the project and show info
make stop   # Stop the project
```

> Run `make help` to see all available commands.
